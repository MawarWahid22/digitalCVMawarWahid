PROJECTS

Augmented Reality Mobile App 

Developed a mobile application for biology education that implement Augmented Reality technology on an animated human heart 3D model and also implement gamification concept by incorporating game elements and multiplayer functionality using Android Studio(Java), Firebase database and Autodesk Maya.

Lost & Found App

Lost & Found is an application for students to report lost, stolen or found items. Users can also search for the lost or found item added by others. This application can update the item that was not claimed by the owner within 1 year without coercion to the missing item holders to sell the item.
